# Firefox 버전 업그레이드
- www.mozilla.org/ko 에서 다운로드
- tar jxf 로 압축 해제
- /firefox 의 폴더 위치 변경
    - mv firefox /usr/local
    - bin 에 심볼릭 링크를 만들어준다
        - cd /usr/local/bin
        - ln -s /usr/local/firefox/firefox .
